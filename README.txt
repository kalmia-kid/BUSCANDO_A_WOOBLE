Installation
Descargar archivo ZIP del repositorio
Abrir el proyecto en unity > Add project > selecciona la carpeta del proyecto > abrir con la version requerida
Configurar XR para Meta Quest (Window > Package Manager) 
Instalar XR Plucguin Management. Edit > Project Settings > XR Plugin Management
Plataforma Windows → activar OpenXR
En OpenXR Settings:
Seleccionar Meta Quest Support si aparece
Activar Oculus Touch Controller Profile
Conectar las gafas Oculus Quest
Ejecutar la escena principal 
Compilar el juego: (File > Build Settings) Seleccionar Windows > Build > ejecutar .exe

Usage
Mecanicas principales: Sistemas de Selección y Manipulación del Entorno Virtual
Selección: Raycasting (Puntero Láser) utilizado principalmente para la selección de elementos de interfaz (menús, botones) a distancia.
Manipulación: Interacción Directa con Grab o agarre. El usuario debe mover su controlador/mano virtual hasta la posición del alien y presionar el botón grip o trigger (gatillo)para "atraparlo".

History 
Seguimos la historia de Wooble, un alien perdido en la tierra, y su regreso a su planeta natal. Wooble trata de pasar desapercibido entre los humanos, por eso trata de esconderse o disfrazarse al estar en lugares muy concurridos. La historia y el lore de Wooble se desarrollará más adelante gracias a los diseñadores.
